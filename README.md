# Hackathon 2026 – Survival Project

This project was created at the **Vorarlberg Hackathon 2026**.

---

## Hackathon

The hackathon is an 8-hour competition where teams develop a project, prototype, or innovative idea in a short time. At the end, a jury evaluates all projects and awards prizes.

---

## Features

- **Survival Guide** – Practical tips for survival situations  
- **World Map** – Shows water sources and key locations  
- **Server Integration** – Reduces RAM and CPU usage for smooth performance
